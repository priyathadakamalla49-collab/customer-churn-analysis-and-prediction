# Raw Dataset

This folder contains the original telecom customer churn dataset used in this project.

## Dataset Information
- Dataset: Telco Customer Churn
- Source: IBM Sample Dataset / Kaggle
- Records: 7,043 customers
- Features: Customer demographics, account information, services subscribed, billing details, and churn status.

## Note
This dataset is stored in its original form and has not been modified. Data cleaning, preprocessing, and feature engineering were performed on copies of this dataset, and the processed files are stored in the `data/cleaned` folder.
